This MATLAB package reproduces Figs 2-9 in the following paper:
X. Ma, A. A. Nasir and D. B. Da Costa, "Hybrid Near/Far-Field Communications of RIS-Aided Cell-Free Massive MIMO With Visibility Region Detection," IEEE Open Journal of the Communications Society, vol. 6, pp. 1710-1726, 2025, doi: 10.1109/OJCOMS.2025.3543393.
Download article: https://ieeexplore.ieee.org/document/10891689
How to Run
• Run Figure2 _and_3.m - generates Fig. 2 and Fig. 3
• Run Figure4.m → generates Fig. 4
• Run Figure5.m → generates Fig. 5
• Run Figure6.m → generates Fig. 6
• Run Figure7.m → generates Fig. 7
• Run Figure8.m → generates Fig. 8
• Run Figure9.m → generates Fig. 9
MATLAB R2024a or later is recommended.
